# Java Debugger Tool

This is a simple Java-based project developed to understand and practice debugging concepts in Java. The tool helps in tracing program execution, handling exceptions, and analyzing logical and runtime errors.

## Features
- Identifies common runtime and logical errors
- Uses exception handling for error tracing
- Helps in understanding program flow
- Beginner-friendly structure

## Technologies Used
- Java
- Object Oriented Programming (OOP)
- Exception Handling

## Purpose
This project was created as part of my academic learning to improve my debugging skills and gain hands-on experience with Java programming.
